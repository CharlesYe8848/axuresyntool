# Axure需求管理同步工具
支持采集Axure原型中的需求卡片中的标题及描述内容

支持将需求卡片中的内容同步到禅道系统中

支持将改动信息发送到云之家中，并能@人

# 如何安装
下载release_crx目录下最新版本的axuresyntool.crx文件

浏览器中输入chrome://extensions/，将crx文件拖动至chrome浏览器中安装使用

点击齿轮按钮进行设置必要参数：禅道同步服务地址、禅道地址、禅道账号
